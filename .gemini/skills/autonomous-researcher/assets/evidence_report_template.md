# Evidence Report Template

このファイルは、`autonomous-researcher` スキルを使用する際に、実装提案の冒頭に記述すべきレポートのテンプレートである。

## 調査レポート (Evidence Report)

- **[Source URL]**: (参照した公式ドキュメント、GitHub Issue、RFC等のURL)
- **[Context]**: (何を知るために調査したか。例: Win32 API `CreatePseudoConsole` のエラーコード `0x80070057` の原因調査)
- **[Key Findings]**: 
  - (調査で得られた主要な事実)
  - (技術的な制約や落とし穴)
- **[Constraint Alignment]**:
  - (プロジェクト規約 `conductor/code_styleguides/` との整合性)
  - (採用する設計判断の根拠)
