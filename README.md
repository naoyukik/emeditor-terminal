# EmEditor Terminal Plugin

EmEditor 内で簡易的なターミナル操作（`pwsh.exe`）を実現するプラグインです。

このプラグインは100% バイブコーディングで行われ、AIエージェントによって実装されています。

## 機能

- **セッション開始**: プラグインを実行するとバックグラウンドで `pwsh.exe` が起動します。
- **コマンド送信**: 2回目以降の実行時に「Send Command」ダイアログが表示され、コマンドを入力して送信できます。
- **出力表示**: コマンドの実行結果は EmEditor のアウトプットバーに表示されます。
- **UTF-8対応**: 内部でエンコーディング変換を行っており、日本語の入出力に対応しています。

## インストール

1. `install.ps1` を PowerShell で実行してください。
   ```powershell
   .\install.ps1
   ```
2. EmEditor を再起動してください。

## 使い方

1. EmEditor の「プラグイン」バーからアイコンをクリックするか、割り当てたキーボードショートカットで実行します。
2. カスタムバーが立ち上がり、 `pwsh.exe` セッションが開始されます。

## 開発者向け情報

- **ログ**: `%TEMP%\emeditor_terminal.log` にデバッグログが出力されます。
- **ビルド**: `cargo build --release`
